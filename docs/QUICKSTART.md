# 快速上手指南

> 5 分钟从零到第一个 PR

---

## 场景：从零开始，用 claude-code-init 完成第一个 PR

### 1. 初始化项目

```bash
npx claude-code-init --project-path ./my-project
cd my-project
```

### 2. 启动 Claude Code

```bash
claude
```

### 3. 开发功能

```
帮我实现用户登录接口，包含 JWT 鉴权
```

AI 自动评估复杂度（涉及安全+数据库），自动走 OpenSpec SDD 流程。

### 4. 审查代码

```
审查代码
```

AI 自动加载 code-review 技能，输出分级审查报告。

### 5. 提交

```
/commit
```

### 6. 进阶：过夜任务

```bash
bash scripts/tmux-session.sh scripts/PROMPT.md
```

---

## 常见流程示例

### 开发新功能

1. **描述需求**：`帮我实现用户注册功能`
2. **AI 自动分流**：评估复杂度 → 决定执行模式
3. **按提示操作**：确认方案 → 审查代码 → 提交

### 修复 Bug

1. **描述问题**：`帮我排查登录失败的问题`
2. **AI 自动调试**：根因分析 → 修复方案 → 验证
3. **按提示操作**：审查修复 → 提交

### 代码重构

1. **描述目标**：`重构订单模块，提高可读性`
2. **AI 自动分析**：影响面评估 → 安全重构流程
3. **按提示操作**：测试保护 → 分步重构 → 验证

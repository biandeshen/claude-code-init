# /fix - 系统化错误修复

> **完整指令见**: `.claude/skills/error-fix/SKILL.md`
> 本文件为快捷入口，完整修复流程（含熔断规则、格式模板等）由对应 Skill 提供。

系统化错误修复：分析→根因→方案→验证→提交闭环。

1. 收集错误信息（终端输出/测试报告/日志）
2. 根因分析（先提假设，验证前不得修改代码）
3. 方案对比（至少两种方案，评估风险）
4. 执行修复并验证（测试通过）
5. 调用 `/commit` 提交

> 熔断规则、输出格式模板等详见 error-fix Skill。

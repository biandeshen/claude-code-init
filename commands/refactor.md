# /refactor - 安全重构

> **完整指令见**: `.claude/skills/safe-refactoring/SKILL.md`
> 本文件为快捷入口，完整重构流程（含检查清单、覆盖率对比等）由对应 Skill 提供。

安全重构，不改变外部行为：

1. 确定边界（目标模块、公共接口、调用方）
2. 编写保护性测试，建立安全网
3. 逐步重构，每步保持测试绿灯
4. 全量测试验证 + 行为对比
5. 调用 `/commit` 提交

> 重构检查清单（前/中/后）、覆盖率对比等详见 safe-refactoring Skill。

## 快速参考

- **功能**：安全重构代码，保持外部行为不变（保护性测试+逐步重构+行为对比）
- **适用场景**：代码整理优化时、消除技术债时、提升可维护性时
- **完整指令**：见 `.claude/skills/safe-refactoring/SKILL.md`

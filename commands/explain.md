# /explain - 深度代码解释

> **完整指令见**: `.claude/skills/code-explain/SKILL.md`
> 本文件为快捷入口，完整解释框架（含动态分析、适用场景矩阵等）由对应 Skill 提供。

解释代码的架构、数据流和关键决策：

- `/explain <文件路径>` — 解释整个文件
- `/explain <文件路径> <函数名>` — 解释特定函数
- `/explain <文件路径> :<行号范围>` — 解释特定代码段

输出：概览 → 架构定位 → 数据流 → 关键设计决策 → 潜在风险和改进建议

> 动态分析维度、适用场景矩阵等详见 code-explain Skill。

## 快速参考

- **功能**：深度解析代码的架构定位、数据流和关键设计决策
- **适用场景**：理解陌生代码时、审查复杂逻辑时、需要代码走查时
- **完整指令**：见 `.claude/skills/code-explain/SKILL.md`

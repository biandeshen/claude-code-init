# 任务执行计划模板

> 本文件为 Claude Code 在处理复杂任务时必须遵循的记录模板。
> 实际执行时，应在项目根目录创建 `Plan.md` 并按照此结构填写。

---

## 这个文件与 OpenSpec 的关系

- OpenSpec 管"事前规划"（Propose→Spec→Design→Task→Check）——在 `openspec/changes/` 下
- 这个 Plan.md 管"事后记录"（执行日志）——在本项目根目录
- **不要用 Plan.md 做事前设计，那是 OpenSpec 的活**

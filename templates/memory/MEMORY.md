## ID 命名规范

| 类型 | ID 前缀 | 示例 |
|------|---------|------|
| 架构决策 | ADR | ADR-001 |
| Bug 记录 | BUG | BUG-001 |
| 编码模式 | PAT | PAT-001 |
| 领域知识 | CTX | CTX-001 |

`access_count` 表示该记忆被检索次数（GC 依据：>90 天且 access_count = 0 → 归档）。

## 归档

超过 90 天且 access_count = 0 的记忆 → 移至 [archive/](archive/) 目录。

> 使用 `/remember` 浏览/编辑/删除记忆。运行 `/gc` 清理过期记忆。

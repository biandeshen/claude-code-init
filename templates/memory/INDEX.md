## ID 命名规范

每条记忆使用 `<前缀>-<编号>` 格式，前缀按主题文件区分：

| 主题文件 | ID 前缀 | 示例 |
|----------|---------|------|
| decisions.md | ADR | ADR-001 |
| bugs.md | BUG | BUG-001 |
| patterns.md | PAT | PAT-001 |
| context.md | CTX | CTX-001 |

`access_count` 表示该记忆被检索/引用的次数（初始为 0），GC 时用于判断是否可归档（>90 天且 access_count = 0 → 归档）。
